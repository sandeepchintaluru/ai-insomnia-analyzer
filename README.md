# AI-Based Insomnia Risk Assessment

## Overview

This project uses Machine Learning to predict insomnia risk based on lifestyle and sleep-related factors.

## Features

* Predicts insomnia risk (Low, Medium, High)
* Uses Random Forest Classifier
* Flask-based web application
* Interactive user interface
* Provides sleep health recommendations

## Input Parameters

* Age
* Sleep Hours
* Screen Time Before Bed
* Caffeine Consumption
* Exercise Days Per Week
* Stress Level
* Daytime Naps
* Irregular Sleep Schedule

## Technologies Used

* Python
* Flask
* Pandas
* Scikit-learn
* HTML/CSS/JavaScript

## Machine Learning Model

Random Forest Classifier
